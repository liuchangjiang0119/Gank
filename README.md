#API
[拿着Api去玩耍](http://gank.io/api)

#实现功能

* 简单的推送展示和搜索功能
* MVP框架
* Retrofit+Rxjava+Glide
* X5WebView
* 移动数据库GreenDAO实践
* OkHttp缓存
* 图片下载


本App主要用于学习和进的技术的实践和应用。
